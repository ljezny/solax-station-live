SOLAR STATION Live is a monitoring station for inverters SOLAX X3-HYBRID G3/G4, GoodWe ET a SOFAR SOLAR HYD.

It displays live data from the inverter:

  • energy flow - clearly shows panel output, household consumption, battery status, and even inverter temperature
  • statistics - how much energy was produced, consumed, purchased, and sold today
  • graph - production, consumption, and battery status for the last 24 hours
What can be set up, can break.

That’s why SOLAR STATION Live is designed so the user doesn’t have to set up anything. The station automatically connects to your inverter’s WiFi and starts displaying data.

Before purchasing

Ensure that there is an available WiFi network from the inverter at the location where you want to use the station:

  • SOLAX - Only installations with a WiFi Dongle are supported. The Dongle broadcasts WiFi named WiFi_XXXXXX, where XXXXXX is the serial number of the dongle. This network must be unsecured by a password - the default state. If such a network is not available, it may still be hidden - this can be verified and set up in the SolaxCloud mobile app.
  • GoodWe - The GoodWe inverter broadcasts a WiFi network named Solar-WifiXXXXXX, where XXXXXX is the serial number of the dongle. This WiFi is normally secured by a password 12345678. Please check that you can connect to this network using that password.
  • SOFAR SOLAR - The SOFAR SOLAR inverter broadcasts a WiFi network named AP-XXXXXX, where XXXXXX is the serial number of the dongle. This WiFi network is typically secured with a password printed on the WiFi dongle. Please ensure you can connect to this network using that password.

In the first version, only typical household photovoltaic (PV) installations with the listed inverters and battery support will be supported. This means multi-inverter installations will not be supported yet (for now).

If you are unsure, contact me via the form.

Installation

Installation is very simple. Just plug the station into the socket using the included cable. The station will automatically connect to the inverter and start displaying data right away. That's it!

Firmware update

SOLAR STATION Live does NOT come with automatic firmware updates, unlike other projects. The device is designed in such a way that if it works now, it will still work in 5 years. However, firmware updates can still be performed using a USB-C cable and a browser.

Firmware update tutorial is here.

SOLAR PLUG

All sold SOLAR PLUG sockets are supported. After connecting the SOLAR PLUG, SOLAR STATION Live will automatically pair with it and a new tile will appear. The sockets are automatically switched according to internal rules to maximize the use of solar energy.
