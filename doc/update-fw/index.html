Aktualizace Solar Station Live přes USB

Pro aktualizaci budeš potřebovat:

  • Počítač s Windows nebo Mac
  • USB-C kabel (datový). Pozor některé kabely jsou pouze nabíjecí.
  • Prohlížeč Chrome, Microsoft Edge

Na počítači si nejprve nainstaluj prohlížeč Chrome nebo Microsoft Edge. V jiném prohlížeči tento postup nebude fungovat.

Postupuj následovně:

  1. Propoj Solar Station a počítač pomocí USB-C kabelu
  2. Tuto stránku otevři v prohlížeči.
  3. Klikni na tlačítko Connect níže.
  4. Po stisku Connect budeš vyzván k výběru sériové linky - pravděpodobně bude dostupná pouze jedna, zvol ji. Bude mít v názvu "usb serial".
  5. Následně zvol Install Solar Station Live. Dojde k vymazání a instalování nového firwmare. Operace trvá několik minut.
  6. Po aktualizaci odpoj a připoj znovu.
