Aktualizace Solar Station Live přes USB

Pro aktualizaci potřebujete:

  • Počítač s Windows nebo Mac
  • USB-C kabel (datový). Pozor některé kabely jsou pouze nabíjecí.
  • Prohlížeč Chrome, Microsoft Edge

Postup:

  1. Propojte Solar Station a počítač pomocí USB-C kabelu
  2. Klikněte na tlačítko Connect níže.
  3. Po stisku Connect budete vyzváni k výběru sériové linky - pravděpodobně bude dostupná pouze jedna, zvolte ji. Bude mít v názvu "usb serial".
  4. Následně zvol Install Solar Station Live. Dojde k vymazání a instalování nového firwmare. Operace trvá několik minut.
  5. To je vše!
